# evolution

gcc main.c $(sdl2-config --cflags --libs)


Evolution is a personal project aimed at developing my skills in 2D graphical interfaces, neural networks, evolutionary algorithms, physical simulations, etc.
Its main objective is to create an evolving animation of simple little beings exploring a basic environment.

Actual version: V0.2 // Actual Milestone: muscle implementation

Roadmap:

v 0.1 :
Window creation,
structural implementation,
dot and line functions,
creature implementation,
simulation running,
basics physics functions,

v 0.2 :
muscle implementation,
neural network implementation,
physics and performances improvements,

v 0.3 :
environment creation,
interface upgrade/interactivity(pause mode, map exploration),
creation of goals for creatures,

v 0.4 :
evolutionary algorithms,
fast training mode without GUI,

v 0.5 :
surely patchs and upgrades
